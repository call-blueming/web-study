# 1. 定位

```javascript
#1.使用场景
不规则的布局

#2.分类
1. 相对定位: relative 
2. 绝对定位: absolute
3. 固定定位: fixed

#3.区别
1) 是否脱离文档流
    1. 相对定位: 否
    2. 绝对定位: 是
    3. 固定定位: 是
    
2) 是否随着滚动条一起滚动
	1. 相对定位: 是
    2. 绝对定位: 是
    3. 固定定位: 否
    
3) 相对定位点
	1. 相对定位: 相对自己
    2. 绝对定位: 相对于最近定位的父辈元素, 如果父辈所有都没有定位-相对于网页文档定位
    3. 固定定位: 相对于浏览器窗口
    
#4.定位层级
	z-index: 100;  数字越大层级越高  必须配合定位一起使用
```
